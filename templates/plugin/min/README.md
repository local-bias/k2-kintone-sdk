# 📦 kintone プラグインテンプレート

[ホームページ](https://ribbit.konomi.app)

## 🔧 使い方

1. 各種ライブラリをインストールします

```
npm install
```

2. プラグインの秘密キーを作成と、SSL 証明書の作成を行います

```
npm run init
```

3. 各種設定を行います(詳細は後述)

4. プラグインを作成し、ご利用の Kintone へアップロード + ファイルの変更を監視

```
npm run dev
```

5. リリース用 zip ファイルの生成

```
npm run build
```
